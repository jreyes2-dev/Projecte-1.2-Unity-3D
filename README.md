**Parkour United** is a 3D platformer developed in **Unity** using **C#**, focused on parkour-based movement and level navigation.

This project was created as part of my game development training, showcasing skills in gameplay programming, level design, and Unity integration.

---

## Overview

The player navigates through different levels by jumping across platforms and avoiding falls. Each fall results in losing a life and restarting the level. Losing all lives returns the player to the main menu.

In later stages, **collectibles** are required to unlock progression.

---

## Features

- 3D parkour-based platforming gameplay  
- Life system with level reset  
- Collectible-based progression  
- Minimalist HUD  
- Main menu, options menu, and pause system  
- Multiple environments with increasing difficulty  

---

## Controls

| Action | Key |
|--------|-----|
| Move | WASD |
| Jump | Space |
| Pause / Menu | ESC |

---

## Levels

- **Level 1 – Lava World**  
  Hazardous environment with structures leading to the next stage  

- **Level 2 – Mountain**  
  Open area with access to an abandoned mine  

- **Level 2 – Mine**  
  Dark environment with floating platforms and increased difficulty
